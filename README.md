# Netflix Exploratory Data Analysis (EDA)

## Objective
To understand data patterns, feature behavior, and insights from Netflix Movies and TV Shows dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Steps
- Data loading and cleaning
- Univariate and multivariate analysis
- Outlier detection
- Correlation analysis
- Automated insight generation
